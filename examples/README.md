# Gator Example Scripts

This directory contains example scripts demonstrating how to use various Gator platform APIs and features.

## Available Examples

| Example | Description |
|---------|-------------|
| `example_per_site_nsfw_filtering.py` | Demonstrates per-site NSFW content filtering configuration |
| `examples_local_image_generation.py` | Shows how to use local image generation capabilities |

## Usage

These examples are meant to be studied and adapted for your own use cases.

```bash
# Run an example
python examples/example_per_site_nsfw_filtering.py
python examples/examples_local_image_generation.py
```

## Prerequisites

Ensure you have installed dependencies:

```bash
pip install -e .
python setup_db.py
```
