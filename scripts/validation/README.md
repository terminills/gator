# Gator Validation Scripts

This directory contains validation and verification scripts used to verify that specific features and implementations are working correctly.

## Scripts

| Script | Description |
|--------|-------------|
| `validate_code_structure.py` | Validates codebase structure and organization |
| `validate_enhancements.py` | Validates enhancement implementations |
| `validate_enhancements_v2.py` | Enhanced validation for implementations |
| `validate_fixes.py` | Validates bug fix implementations |
| `validate_ipmi_implementation.py` | Validates IPMI feature implementation |
| `validate_nsfw_filtering.py` | Validates NSFW filtering functionality |
| `validate_sdxl_long_prompt.py` | Validates SDXL long prompt handling |
| `verify_ai_model_detection.py` | Verifies AI model detection capabilities |
| `verify_rss_implementation.py` | Verifies RSS feed implementation |
| `verify_sdxl_changes.py` | Verifies SDXL-related changes |
| `verify_fixes.sh` | Shell script for verifying fixes |

## Usage

Run validation scripts from the repository root:

```bash
python scripts/validation/validate_code_structure.py
python scripts/validation/verify_rss_implementation.py
./scripts/validation/verify_fixes.sh
```
